GridLayer

abstract class GridLayer<T : GridLayer.GridLayerOptions>(options: T = definedExternally) : Layer<T> (source)

Generic class for handling a tiled grid of HTML elements. This is the base class for all tile layers and replaces Canvas.

GridLayer can be extended to create a tiled grid of HTML elements like <canvas>, <img> or <div>. GridLayer will handle creating and animating these DOM elements for you.

https://leafletjs.com/reference.html#gridlayer
