open fun invalidateSize(options: LeafletMap.ZoomPanOptions = definedExternally): LeafletMap

Checks if the map container size changed and updates the map if so â€” call it after you've changed the map size dynamically, also animating pan by default. If InvalidateSizeOptions.pan is false, panning will not occur. If InvalidateSizeOptions.debounceMoveend is true, it will delay moveend event so that it doesn't happen often even if the method is called many times in a row.
